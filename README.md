# NYC Airbnb Data: Advanced Cleaning & Preliminary Insights
NYC Airbnb Data: Advanced Cleaning &amp; Preliminary Insights. Focuses on data quality, handling outliers (IQR), and neighborhood-based trends.

## 📌 Project Overview
As a 16-year-old data enthusiast, I dedicated this project to the most critical step in data science: **Data Quality**. Using Python and Pandas, I performed a deep clean of the NYC Airbnb dataset to extract reliable, noise-free insights about the rental market.

## 🛠️ What I Achieved (The Cleaning Process)
* **Handling Missing Data:** Identified and managed null values to ensure dataset integrity.
* **Advanced Outlier Detection:** Implemented the **Interquartile Range (IQR)** method to filter out unrealistic price points (e.g., $0 or $10,000+ listings), logically capping the analysis at **$333**.
* **Data Standardization:** Converted date columns and cleaned categorical strings for accurate grouping.

## 📊 Preliminary Insights
By analyzing the cleaned data, I discovered:
* **Market Trends:** Identified the distribution of listings across different room types.
* **Neighborhood Analysis:** Found that **Brooklyn** and **Manhattan** show the highest activity, with specific trends in availability and pricing after removing extreme outliers.

## 🚀 Why This Matters
This project proves that "Clean Data" is the foundation of any real-world analysis. My focus was on transforming raw, "dirty" data into a trustworthy source for decision-making.
